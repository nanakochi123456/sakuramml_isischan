  • MMF=SMAF形式は、ヤマハ株式会社の提唱している着信メロディーフォーマットです。主にau、Softbankで使用できます。MicroSD等に転送するか、自分宛てに添付メールを送信することで、携帯電話に登録することができます。
  • 128音着メロをPCで再生するには、ATS-MA7-SMAF内のSMAF Player Sが必要です。ミッドラジオプレイヤーがインストールされている場合、MMF拡張子が関連付けられていますが、MA-5以前しか対応していないため、再生できません。
